## Call for Participation
This roundtable fosters dialogue about the current state of digital methods for Internet research in Brazil. We seek to celebrate emerging research practices and kick off a Global South network, situating them within a transitional methodological moment in which digital methods and methodologies have been built with, in and about AI, web platforms and data visualisation. This roundtable does not provide an exhaustive overview of digital methods in Brazil.
